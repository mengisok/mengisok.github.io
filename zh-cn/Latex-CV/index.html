LaTeX 简历

10月 29 2022 工作 4 分钟 读完 (约 542 字)

从昨天中午到今天中午，我一直在用latex做简历。这对初学者来说非常不友好😢。幸运的是，我现在做完了，看看我的简历.

我是这样做的：

-> 如果你只是偶尔或少量使用latex，一个在线编辑器 Overleaf 可以省去很多麻烦。你可以找模板，然后编辑和编译就可以了。
-> 如果你想做更多的事情，请使用以下步骤。

下载 MiKTeX 和 VSCode

MiKTeX
你会得到MiKTeX Consle和TeXworks。后者是一个编辑器，但我更喜欢使用VSCode，因为它可以在另一个窗口预览pdf（也因为我已经有VSCode）。
另一个选择是Tex Live。它预装了一个完整的软件包，但下载速度太慢，而且经常失败，所以我选择了MiKTeX。

VSCode
记得下载插件 latex workshop。

找模板

在Overleaf或Github上找一个模板。
这是我喜欢的：
新卒のためのLaTeX履歴書テンプレート
TLC-Resume-2021
以及我做的：
Resume-CV (在TLC-Resume-2021的基础上修改的)

编辑模板

在这一步你可能会遇到很多问题，其中大部分需要你在MiKTeX控制台中下载包，或其他东西，如Perl。只要按照错误信息中所说的安装缺少的项目就可以了。

分享模板

  1. 把你的模板放到github上
    (已经绑定了git和你的github账户并设置了SSH密钥)
    -> 创建一个新的仓库
    -> Git bash文件夹，其中包含关于你的模板的全部文件
    -> 在Git中运行以下代码
git init
git add .
git commit -m 'Description'
git remote add origin https://...  (copy from github)
git push -u origin master
  1. 提交到Overleaf
    我就是在那里找到我的简历的原始模板的! 我想，把模板放在那里可能会帮助到更多的人。
