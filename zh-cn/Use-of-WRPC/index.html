WRPC的使用

11月 4 2023 实验室工作 6 分钟 读完 (约 883 字)

简单总结了一下最近使用White Rabbit PTP Core(WRPC)的经验。

软件和硬件准备

在OHWR下载所需的文件（版本为4.2）：

  • wr-cores （HDL代码）
  • wrpc-sw （LM32软件代码）
  • wrpc-user-mannual （使用手册）

硬件包括可运行WRPC的PCB，有Linux系统的电脑，WR交换机，光纤等。

1. 准备Linux环境
对于Windows系统来说，可以安装子系统Windows Subsystem for Linux (WSL)。

两个建议：

  • 设置WSL的默认版本为1。WSL 2似乎有bug，可能安装失败。
  • 不要下载太新的Linux版本，以免遇到版本不匹配的问题，毕竟wrpc-v4.2是2019年的。我用的是Ubuntu LTS 18.04。

2. 交叉编译
wrpc-sw为C代码，而FPGA上的LM32 CPU需要用HDL（IP核即可），因此需要交叉编译，将C代码转换为适合于 LM32 CPU的二进制文件。

使用WRPC

1. 添加WRPC
新建一个Vivado项目，写好测试文件，将wr_core.v和wr_core.edif添加到项目中。

2. 生成bit文件和ltx文件
点击Generate Bitstream。

3. 烧写前的准备
连接：

  • JTAG连接：通过JTAG下载器连接PC和FPGA
  • 串口连接：通过USB Type-C连接PC和FPGA（PuTTY教程）
  • 光纤连接：通过光纤连接WR交换机（主节点）和PCB（从节点）

FPGA和WR交换机均需连接电源。

4. 烧写逻辑，调试
点击Open Hardware Manager -> Auto Detect -> Program Device，将bit文件和ltx文件烧写进FPGA。可以用ILA IP核调试FPGA。

在PuTTY Shell中按回车健，显示 wrc# 就说明WRPC正在运行了。wrpc-user-mannual的附录A中列出了WRPC Shell命令，不妨先试试 gui 和help 。
